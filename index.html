Aprenda a usar a internet com segurança

Este guia foi criado para ajudar você a navegar na internet, utilizar o Pix e identificar golpes digitais.

Módulos de Aprendizado

O que é internet?

A internet é como uma grande rede que conecta pessoas.

  • Conversar pelo WhatsApp
  • Assistir vídeos
  • Pagar contas
  • Pesquisar informações

⚠️ Mas é preciso cuidado ao clicar em links desconhecidos.

Como usar o Pix com segurança

O Pix é uma forma rápida de transferir dinheiro.

Antes de confirmar:

  • Verifique o nome da pessoa
  • Confira o valor
  • Não faça Pix para desconhecidos

🚫 Bancos não pedem senha por mensagem.

Como identificar golpes

Golpes tentam enganar você para roubar dinheiro ou dados.

Fique atento se a mensagem:

  • Promete dinheiro fácil
  • Pede sua senha
  • Cria urgência: "Última chance!"

Se tiver dúvida, não clique. 🚫

Como proteger seus dados

  • Não compartilhe sua senha
  • Use senhas diferentes
  • Ative verificação em duas etapas
  • Evite usar Wi-Fi público para banco

Teste seus conhecimentos

1. Um banco pede sua senha ou código por mensagem de WhatsApp. Isso é seguro?

2. Recebi uma notícia bombástica de um amigo no WhatsApp sem link de fonte confiável. Devo compartilhar?

3. Usar a mesma senha (como a data de nascimento) para tudo é uma boa prática?

4. Um anúncio diz que ganhei um prêmio incrível de uma loja famosa, mas pede para clicar num link estranho. O que fazer?

5. Manter o celular e os aplicativos atualizados ajuda na segurança do aparelho?
